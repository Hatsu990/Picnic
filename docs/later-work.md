# 추후 작업 메모

## Solapi 문자

- 목적: 예약 접수/확정 안내 문자와 홍보 문자를 발송할 수 있게 만든다.
- 현재 범위: API 키 환경변수 자리만 유지하고, 실제 발송 자동화는 아직 연결하지 않는다.
- 나중에 필요한 것: 발신번호 등록, 광고 문자 수신 동의, 야간 발송 제한, 수신 거부 문구, 발송 로그 테이블.

## 마케팅 자동화

- 목적: 예약 고객에게 쿠폰/재방문 메시지를 자동 발송한다.
- 현재 범위: 구현하지 않고 후순위로 둔다.
- 나중에 필요한 것: 마케팅 수신 동의, 발송 대상 세그먼트, 쿠폰 코드 테이블, Solapi 발송 큐, 실패 재시도, 발송 이력 화면.

## 결제/운영

- 결제는 아직 붙이지 않는다.
- 운영자 메뉴 CRUD, 배포 환경 QA, 실제 운영 env 정리는 네이버 로그인/예약 조회 이후에 진행한다.
